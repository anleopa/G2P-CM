Gene

Description

A chromosome element that includes all of the sequence elements necessary to encode a functional transcript

References

  • Sequence Ontology: SO:0000830

Attributes

Name Type Description
name String

Official name used as consensus to describe the gene
symbol String

Official symbol used as consensus to identify the gene, usually an acronym of the name
biotype String

The gene classification, usually according to its biological function

  • Chromosome Element

Sources and Tools

NCBI Gene: Gene integrates information from a wide range of species. A record may include nomenclature, Reference Sequences (RefSeqs), maps, pathways, variations, phenotypes, and links to genome-, phenotype-, and locus-specific resources worldwide.
HGNC: The HGNC is responsible for approving unique symbols and names for human loci, including protein coding genes, ncRNA genes and pseudogenes, to allow unambiguous scientific communication.
GeneCards: Searchable, integrative database that provides comprehensive, user-friendly information on all annotated and predicted human genes. The knowledgebase automatically integrates gene-centric data from ~150 web sources, including genomic, transcriptomic, proteomic, genetic, clinical and functional information.
