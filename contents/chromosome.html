Chromosome

Description

Structural unit composed of a nucleic acid sequence which controls its own replication

References

  • NCBI: https://www.cancer.gov/publications/dictionaries/genetics-dictionary/def/chromosome

Name of the chromosome
sequence String

Sequence of nucleotides that conforms the chromosome
reference_genome String

Identifier of the reference genome used as consensus and derived from the assembled sequences of different donors.
