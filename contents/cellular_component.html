Cellular Component

Description

The locations relative to cellular structures in which a gene product performs a function, either cellular compartments (e.g., mitochondrion), or stable macromolecular complexes of which they are parts (e.g., the ribosome). A cellular component refers not to processes but rather a cellular anatomy.

References

  • Gene Ontology: http://geneontology.org/docs/ontology-documentation/

Attributes

Name Type Description
id String

Value that univocally identifies the cellular component in a database
name String

Name of the cellular structure of which the gene product is part of
description String

Detailed description of the cellular component
