Observable Characteristic

Description

Sign or characteristic resulting from the expression of genes and liable to be observed; the clinical presentation of an individual with a particular genotype

References

  • NCI: https://www.cancer.gov/publications/dictionaries/genetics-dictionary/def/phenotype

Attributes

Name Type Description
id String

Value that univocally identifies the observable characteristic in a database
name String

Name commonly used to represent the observable characteristic

Detailed description of the observable characteristic
