Takes part

Description

This class represents the role that a participant in an event has.

Attributes

Name Type Description
role String

Type of role that the participant has in the event. The common roles are input, output or catalyst in case of enzymes.

Relationships

  • (Protein—Event)
  • (Molecular Component—Event)
  • (ncRNA—Event)

Sources and Tools
