Variant

Description

An alteration that occurs in a biological sequence whose extent is the deviation from what is found in most people (sequence of reference).

References

  • Sequence Ontology: SO:0001060; SO:0001059; SO:0000110
  • GHR: https://ghr.nlm.nih.gov/primer/mutationsanddisorders/genemutation
  • NCI: https://www.cancer.gov/publications/dictionaries/genetics-dictionary/def/genetic-variant

Attributes

Name Type Description
id String

Value that univocally identifies the variant in a database
name String

Name commonly used to represent the variant in the literature (e.g. a HGVS expresion or protein change)

reference_allele String

Biological sequence that is found in most people (sequence of reference) in the position where the variant is located
alternative_allele String

Biological sequence that differs from the sequence of reference in the position where the variant is located
start Long

Genome position where the variant begins
end Long

Genome position where the variant ends
clinical_significance String

Variant classification according to its importance for clinical practice (e.g. pathogenic, benign, etc.)

Sources and Tools

ClinVar: Database which aggregates information about genomic variation and its relationship to human health.
Ensembl: Genome browser for vertebrate genomes that supports research in comparative genomics, evolution, sequence variation and transcriptional regulation. Ensembl annotate genes, computes multiple alignments, predicts regulatory function and collects disease data.
AlzForum Mutations: repository of variants in genes linked to Alzheimer’s disease (AD).
