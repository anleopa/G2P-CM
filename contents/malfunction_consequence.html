Malfunction Consequence

Description

Consequence derived from the fail of a pathway or cellular component to work or operate correctly in a specific tissue or organ that derives in a set of observable characteristics

Attributes

Name Type Description
name String

Name used to describe the consequence
description String

Detailed description of the consequence
