Molecular Effect

Description

Effect upon molecular function that a DNA variant can cause on a transcript with respect to a reference (usually a normal function)

Attributes

Name Type Description
name String

Name of the molecular effect (e.g. Missense, Frameshift, etc.)
impact String

Assessment of the putative impact of the variant (e.g. High, Moderate, Low, etc.)
functional_consequence String

The functional consequence produced by the molecular effect (e.g. loss of function, gain of function, etc.)
description String

Detailed description of the molecular effect

Sources and Tools

SnpEff: Genomic variant annotations and functional effect prediction toolbox.
VEP: Tool to determine the effect of variants (SNPs, insertions, deletions, CNVs or structural variants) on genes, transcripts, and protein sequence, as well as regulatory regions.
PoliPhen-2: Tool which predicts possible impact of an amino acid substitution on the structure and function of a human protein using straightforward physical and comparative considerations.
SIFT: Tool which predicts whether an amino acid substitution affects protein function based on sequence homology and the physical properties of amino acids. SIFT can be applied to naturally occurring nonsynonymous polymorphisms and laboratory-induced missense mutations.
