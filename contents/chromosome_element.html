Chromosome Element

Description

A biological region of a chromosome, defined by its disposition to be involved in a biological process

References

  • Sequence Ontology: SO:0000830; SO:0001411; SO:0000001

Attributes

Name Type Description
id String

Value that univocally identifies the chromosome element in a database
start Long

Genome position where the chromosome element starts
end Long

Genome position where the chromosome element ends
strand String

The chain of the chromosome that indicates the direction in which the element is transcribed, commonly named as forward (3’ to 5’) and complement (5’ to 3’).
