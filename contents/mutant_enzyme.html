Mutant Enzyme

Description

Enzyme which function is altered by the molecular effect of a variant

Generalized Elements

  • Enzyme

Specialized Elements

  • Mutant Element

Attributes

Name Type Description
reference_aa String

Sequence of aminoacids that is found in most people (sequence of reference) in the position where the variant is located
alterantive_aa String

Sequence of aminoacids that differs from the sequence of reference in the position where the variant is located
start Long

Position relative to the protein sequence where the variant begins
end Long

Position relative to the protein sequence where the variant ends
